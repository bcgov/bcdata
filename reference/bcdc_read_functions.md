# Formats supported and loading functions

Provides a tibble of formats supported by bcdata and the associated
function that reads that data into R. This function is meant as a
resource to determine which parameters can be passed through the
`bcdc_get_data` function to the reading function. This is particularly
important to know if the data requires using arguments from the read in
function.

## Usage

``` r
bcdc_read_functions()
```
